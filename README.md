# TravelMate_Gr24
Aplikacion për Udhëtime &amp; Eksplorim

# 1️⃣ Klono projektin
git clone https://github.com/LeonitaRama/TravelMate_Gr24.git

# 2️⃣ Hyr në direktoriumin e projektit
cd travelmate

# 3️⃣ Instalo varësitë
npm install

# 4️⃣ Instalo Expo Router (nëse nuk është i instaluar)
npx expo install expo-router react-native-safe-area-context react-native-screens

# 5️⃣ Nis aplikacionin
npx expo start
